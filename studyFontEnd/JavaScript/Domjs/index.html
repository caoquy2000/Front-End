Bài 8:
        Dùng các phương thức Javascript trong DOM (tạo, xoá, sửa các phần tử
        DOM) để tạo và chỉnh sửa động một HTML table.
        Yêu cầu:
        o Nút tạo mới 1 HTML table (2x2)
        o Nút thêm 1 hàng vào HTML table có sẵn
        o Nút thêm 1 cột vào HTML table có sẵn
        o Nút xoá 1 hàng (1 cột) khỏi HTML table có sẵn (chỉ số hàng/cột cần xoá
        được nhập vào bởi
        người dùng thông qua phần tử input text)
        o Nút xoá toàn bộ HTML table có sẵn
