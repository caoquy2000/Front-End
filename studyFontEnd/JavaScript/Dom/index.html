DOM tree




Change the content of html

In the DOM, all HTML elements are defined as objects.
A property is a value that you can get or set (like changing the content of an HTML element).
A method is an action you can do (like add or deleting an HTML element).

Element

Properties

  • attributes: Trả về attribute của el đó
  • children: Trả về collections children của el đó (ko bao gồm cả text và comment)
  • classList: trả về các class của el
  • className: giá trị class của el đó
  • clientHeight/ clientWidth: Trả về chiều cao/ chiều rộng của el (bao gồm padding)
  • clientLeft/ clientTop: Trả về độ dày border top/left của el
  • firstChildElement/lastChildElement: Trả về el con đầu/cuối
  • innerHTML: Nội dung của el
  • previousElementSibling/nextElementSibling: el kế cận phía trước/sau
  • nodeValue: value của một node Tìm hiểu node là gì!
  • offsetHeight/offsetWidth: chiều cao/rộng của el, tính cả border, padding và scrollbar
  • offsetTop/ offsetLeft: vị trí của el so với thằng cha có position khác static (tính vị trí, margin của nó + padding, border, margin, scrollbar của thằng cha)
  • offsetParent: trả ra thằng cha đó
  • parentElement: trả về thằng cha của el
  • scollHeight/scrollWidth: Trả về chiều cao/rộng, có thêm padding, điểm khác với clientHeight/clientWidth đó là phần nào dù bị ẩn đi do overflow(tràn) thì vẫn được tính
  • scrollTop/scrollLeft: Giá trị mà thanh scroll đã cuộn theo chiều dọc/ngang trong el mà có chứa scrollbar
  • tabIndex: để xem khi tab thì thứ tự nó là như nào

Method

  • addEventListener/removeEventListener
  • appendChild(node): thêm phần tử con vào cuối
  • focus() => làm cho phần tử đó đc focus, blur() thì ngược lại
  • click() => ...
  • cloneNode(boolean => có cần clone luôn cả những thằng bên trong hay ko?) => tự clone chính nó, sau đó trả về node mới
  • contains(node) => có chứa node đó hay ko? => true/false
  • getBoundingClientRect() => trả về top/right/bottom/left của node đó so với viewport và width, height của chính nó

  • insertAdjacentText(), tương tự trên với text
  • removeChild(node) => remove node con
  • removeAttribute("attr-name") => xóa attr đó khỏi node
  • setAttribute("attr", "value")

Collections là gì? NodeList là gì?

  • getElementsByTagName() sẽ trả ra collections
  • A NodeList object is a list (collection) of nodes extracted from a document. childNodes, querySelectorAll
  • HTMLCollection items can be accessed by their name, id, or index number. NodeList items can only be accessed by their index number. Only the NodeList object can contain attribute nodes and text nodes.
  • you cannot use array methods like valueOf(), pop(), push(), or join() on an HTMLCollection and NodeList
